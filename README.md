# Ephemeris Core
A cryptographic authentication prototype utilizing real-time celestial ephemerides and temporal coordinates for dynamic entropy generation.

Prototype authentication system using planetary distance + SHA-512 with a Ghost Seal (device binding) and Time-Shift PIN.

## Pages
- `index.html` Login
- `register.html` Register
- `welcome.html` One-time device enrollment
- `panel.html` User panel

## Backend
- `login.php` API (SQLite)

## Notes
- Prototype only. Do not use in production.
- TLS is mandatory for any real deployment.
